# Retail Sales Analytics Dashboard

## 📊 Project Overview
Analyzed retail sales data using SQL and Power BI to uncover trends, store performance, and external factors impacting sales.

## 🔧 Tools Used
- MySQL (Data Analysis)
- Power BI (Dashboard)
- Excel/CSV (Data Source)

## 📈 Key Insights
- Peak sales occur in December (seasonal trend)
- Top stores contribute majority of revenue
- Moderate temperature increases sales
- High fuel prices reduce customer demand

## 📌 SQL Analysis
Includes queries for:
- Sales trends
- Store performance
- Holiday impact
- External factor analysis

## 🚀 Outcome
Built an end-to-end data analytics project demonstrating business insights and visualization skills.
